How To Play Mastermind

  • This is a game where a player tries to guess the number combinations.
  • Player must select a number between 0-7.
    • You have an option to change this to make it more challenging. Change the option below.
    • **Note that duplicate numbers could exist in the Mastermind's code.
  • At the end of each attempt to guess the 4 number combinations, the computer will provide feedback whether the player had guess a number correctly, or/and position correctly.
    • '1' -> Number is in the combination and in the CORRECT position.
    • '2' -> Number is in the combination but in the WRONG position.

  • A player must guess the right number combinations within 10 attempts to win the game.
